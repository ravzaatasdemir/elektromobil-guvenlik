# Elektromobil Guvenlik Kontrol Sistemi

## Aciklama
Bu program, bir elektromobilin yaris oncesi guvenlik protokollerini simule eder.
Sensor verileri kontrol edilerek motor calismasina izin verilip verilmeyecegi belirlenir.

## Kurulum ve Calistirma
Derleme:
gcc main.c -o main

Calistirma:
./main
